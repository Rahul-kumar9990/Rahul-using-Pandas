# Data Analytics with Python & Pandas

A structured repository dedicated to mastering core data manipulation, cleaning, and transformation techniques using **Pandas**. This collection serves as a practical, step-by-step guide for turning raw, unformatted datasets into clean, analysis-ready structures.

---

## 🛠 Tech Stack
* **Python**
* **Pandas** (Core library for data structures and manipulation)
* **NumPy** (For underlying numerical arrays and operations)

---

## 📂 Repository Contents

Each Jupyter Notebook (`.ipynb`) covers a specific module of the Pandas ecosystem:

| Filename | Focus Area & Key Methods Covered |
| :--- | :--- |
| `1)Series.ipynb` | Introduction to 1D Pandas Series, indexing, and vector operations. |
| `2)DataFrames.ipynb` | Constructing, indexing, selecting, and modifying 2D DataFrames. |
| `3)Missingdata.ipynb` | Detecting, dropping, and imputing null values (`isnull`, `dropna`, `fillna`). |
| `4)MergingJoiningConcatenation.ipynb` | Combining multiple datasets horizontally and vertically (`concat`, `merge`, `join`). |
| `5)GroupbyAggregation.ipynb` | Splitting data, applying aggregate functions, and summarizing subsets (`groupby`, `agg`). |
| `6)Pivottable.ipynb` | Reshaping data into multi-dimensional summary tables (`pivot_table`, `stack`/`unstack`). |
| `7)Operations.ipynb` | Applying custom functions, sorting, and vectorized string/arithmetic operations. |
| `Countries.ipynb` | Practical application of Pandas workflows on a country-level dataset. |
| `FeatureExtraction.ipynb` | Deriving new analytical columns and transforming variables for downstream tasks. |

---

## 📂 Data Sources
* `Countries.csv`: Real-world geographic and demographic indicators.
* `anime.csv`: Entertainment dataset used for exploratory grouping and feature extraction.
